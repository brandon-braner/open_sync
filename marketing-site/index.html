Everything in one place

Managing configurations across multiple tools is painful. OpenSync fixes this by providing centralized registries and single-click syncing.

Centralized MCP Registry

Add, edit, and remove MCP server definitions in a single local database (SQLite). Pick the targets you care about and push.

Skills & Workflows

Manage custom instructions, system prompts, rule files, and reusable slash-command workflows across all agents.

LLM Provider Sync

Discover, register, and sync LLM API keys and base URLs. Stop pasting your Anthropic/OpenAI keys into every new tool.

Global & Project Scopes

Manage a system-wide set of configurations and per-project overrides. Use specific rules and tools for specific projects.

Auto Backup & Discovery

Detects servers already present in your installed tools. Timestamped backups are created before every write, so nothing is lost.

Awesome Web UI

A fast React-based dashboard to intuitively manage your MCP Servers, Skills, Workflows, and Providers.
