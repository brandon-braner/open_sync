## About OpenSync

OpenSync is an open source tool that syncs MCP (Model Context Protocol) servers, skills, workflows, and LLM providers across AI agents and IDEs — from one place.

**The Problem**: Managing MCP is painful when you use multiple AI tools. Each editor, CLI, and desktop app keeps its own config file in its own format. Add a server in Cursor, then copy-paste it into Claude Desktop, VS Code, Gemini CLI… and repeat every time something changes.

**The Solution**: OpenSync lets you register your MCP servers once, pick the targets you care about, and sync. You can also manage skills (custom instructions/system prompts), workflows (slash-command sequences), and LLM providers across tools — all from a single dashboard.

**Key Features**:
- Centralized MCP server registry (SQLite)
- Centralized skills registry 
- Centralized workflows registry
- LLM provider management
- One-click sync to 10+ AI tools (Cursor, VS Code, Claude Desktop, Claude Code, Gemini CLI, OpenCode, Windsurf, Antigravity, Copilot CLI, Warp)
- Auto-discovery of existing configs
- Project scanner for agent artifacts
- Global & project scopes
- Config backups before every write
- Format translation between JSON/YAML/Markdown
- Two-tier React Web UI

**Target Users**: Developers using multiple AI coding tools who want to manage their MCP configurations in one place.

**GitHub**: https://github.com/brandonbraner/open_sync
